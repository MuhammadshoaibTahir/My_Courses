Curriculum & Weekly Plan

A modular curriculum across three specializations. Each module contains lectures, readings, lab assignments and a mini-project.

Module 1 — Corpus Creation & Annotation
Annotation, tools, metadata, formats
  • Week 1: Corpus design, sampling, ethics
  • Week 2: Annotation schema, inter-annotator agreement
  • Week 3: Tools: ELAN, Praat, WebAnno, INCEpTION
Module 2 — Python for Text
From basics to NLP pipelines
  • Week 4: Python basics, data structures, file I/O
  • Week 5: Tokenization, normalization, regex
  • Week 6: POS tagging, parsing, NLTK & spaCy
  • Week 7: Word embeddings & transformers (Hugging Face)
Module 3 — Portfolio & Deployment
GitHub, demo apps, docs
  • Week 8: Project wrap-up, portfolio assembly, CV & LinkedIn
  • Week 9: Build a demo web app for your project
  • Week 10: Interview prep and publishing
